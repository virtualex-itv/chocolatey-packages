## Usage

To install Omnissa Horizon Client for Windows, run the following command from the command line or from PowerShell:

```powershell
choco install omnissa-horizon-client
```

To upgrade Omnissa Horizon Client for Windows, run the following command from the command line or from PowerShell:

```powershell
choco upgrade omnissa-horizon-client
```

To uninstall Omnissa Horizon Client for Windows, run the following command from the command line or from PowerShell:

```powershell
choco uninstall omnissa-horizon-client
```

## Description

Omnissa Horizon Client for Windows allows you to connect to your Omnissa Horizon virtual desktop from your device of choice giving you on-the-go access from any location.

**Please Note**: This is an automatically updated package. If you find it is out of date by more than a day or two, please contact the maintainer(s) and let them know the package is no longer updating correctly.
