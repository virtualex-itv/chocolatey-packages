## Usage

To install Pangolin, run the following command from the command line or from PowerShell:

```powershell
choco install pangolin
```

To upgrade Pangolin, run the following command from the command line or from PowerShell:

```powershell
choco upgrade pangolin
```

To uninstall Pangolin, run the following command from the command line or from PowerShell:

```powershell
choco uninstall pangolin
```

## Description

Pangolin is an identity-aware VPN and proxy solution for zero-trust remote access to anything, anywhere. Built on WireGuard, it provides secure tunneled access to your self-hosted services and networks.

## Features

- Zero-trust network access (ZTNA)
- WireGuard-based VPN tunneling
- Identity-aware proxy
- Secure remote access to self-hosted services
