## Usage

To install vTUI - vSphere Terminal UI, run the following command from the command line or from PowerShell:

```powershell
choco install vtui
```

To upgrade vTUI - vSphere Terminal UI, run the following command from the command line or from PowerShell:

```powershell
choco upgrade vtui
```

To uninstall vTUI - vSphere Terminal UI, run the following command from the command line or from PowerShell:

```powershell
choco uninstall vtui
```

## Description

vTUI is a terminal-based user interface for browsing and managing VMware vSphere inventory from the command line.

Features include real-time inventory browsing via the vSphere PropertyCollector API, VM power management, full-text resource search, object property inspection, and JSON export of inventory data.
