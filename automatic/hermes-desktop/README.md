# <img src="https://cdn.jsdelivr.net/gh/virtualex-itv/chocolatey-packages@b19e89ab3ebe0f78a9ea5eca8095633f0781901f/icons/hermes-desktop.png" width="48" height="48"/> [hermes-desktop](https://community.chocolatey.org/packages/hermes-desktop)

Hermes Desktop is the native AI agent from Nous Research, providing persistent memory, focused automation, isolated subagents, web browsing, and multi-model reasoning across Telegram, Discord, Slack, WhatsApp, Signal, Email, and CLI - one agent, one memory, every surface.

* **Chat with the full agent** - streaming responses, live tool activity, structured tool summaries, and shared conversation history with the CLI and gateway.
* **Side-by-side previews** - render web pages, files, and tool outputs in a right-hand pane while you keep chatting.
* **File browser, voice, and a real settings UI** - explore the working directory, talk to Hermes and hear it back, and manage providers / models / tools / credentials without dropping to a config file.
* **Stays current** - built-in updates pull the latest agent and rebuild the app in place.

MIT licensed. Paid tiers include monthly credits for the Nous Portal (300+ models and built-in tool use).

**Please Note**: This is an automatically updated package. If you find it is out of date by more than a day or two, please contact the maintainer(s) and let them know the package is no longer updating correctly.
