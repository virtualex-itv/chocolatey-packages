# <img src="https://cdn.jsdelivr.net/gh/virtualex-itv/chocolatey-packages@a26844fe3b8da7db899f9b024f4b17c61c2aefd2/icons/vtui.png" width="48" height="48"/> [vtui](https://community.chocolatey.org/packages/vtui)

vTUI is a terminal-based user interface for browsing and managing VMware vSphere inventory from the command line.

Features include real-time inventory browsing via the vSphere PropertyCollector API, VM power management, full-text resource search, object property inspection, and JSON export of inventory data.

**Please Note**: This is an automatically updated package. If you find it is out of date by more than a day or two, please contact the maintainer(s) and let them know the package is no longer updating correctly.
